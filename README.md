***Pure C++ Text Editor
A Modern-ish Text Editor using purely C++ and WinApi

**Features
* Spell checker.
* Search and Highlight.
* Auto saving every 60 seconds.
* The ability to Control the number of Colomns, rows and charachters per rows.
* Text Alignment
* Word Integrity across lines
